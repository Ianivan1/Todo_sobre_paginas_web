Que son las paginas web


¿Qué es una página web?

Es sumamente común encontrarse los conceptos de página web y sitio web utilizados de forma indiferente. Pero, ¿sabías que no significan exactamente lo mismo? Para evitar que caigas en este común error te vamos a contar cuál es la diferencia entre estos dos populares conceptos. Una vez que leas este artículo, no los confundirás nunca más. ¡Promesa! Lo primero que debes saber es que cuando nos referimos a un sitio web (o website) estamos pensando en un gran espacio alojado en un servicio de hosting, y que a su vez contiene una gran cantidad de documentos asociados (páginas), y organizados de forma jerárquica. No existe un límite de páginas que pueden estar asociadas a un sitio web, por eso, cada vez que entramos en un sitio como Wikipedia nos sorprende la gran cantidad de contenido que alberga. Ahora bien, y ¿qué es una página web, entonces? Tranquilo, es muy sencillo. Una página web es un documento HTML con información, que puede ser textual, visual, sonora o gráfica y a la que se puede acceder a través de HTTP. Un conjunto de páginas asociadas y ordenadas jerárquicamente conforman un sitio web. Todo su contenido se encuentra en formato HTML y, en un mismo sitio, la estructura de cada página web debe ser coherente. ¿Has visto qué fácil es diferenciarlas?


Partes de una página web

Como ya te dijimos, una página web es un espacio optimizado para que podamos acceder a ella desde cualquier navegador. En ella pueden incluirse textos, imágenes y también enlaces a otras páginas. Cabe mencionar que una página web se caracteriza por plantear una organización bien definida. A continuación te contamos muy brevemente cuáles son las partes de una página web. Una página web tiene: * Cabecera (o header). En ella se plantea la información relevante respecto a la marca: nombre, logo, lema, etc.; * Cuerpo (o body). Es la parte que alberga el contenido que da sentido a la página: artículos, fotografías, etc.; * Pie de página (o footer). Se encuentra al final de la página y en él se muestran generalmente los métodos de contacto, los botones de las redes sociales y un pequeño menú para navegar fácilmente por el sitio. En todo sitio web el total de sus páginas comparten cabecera y pie de página, mientras que el cuerpo es distinto en todas sus páginas. Saber esto te puede servir para conocer lo básico a la hora de plantearte el desarrollo de una página web.
